# Maple Software — Symbolic Mathematics Environment

## About Maple Software

Maple Software provides a complete symbolic mathematics environment for exact solving, calculus operations, differential equations, and matrix manipulation. Maple Software combines a document interface, 2D/3D plotting, and code export. Maple Software supports integration, differentiation, linear algebra, and numeric approximation directly into the interface. Maple Software integrates optimization packages, statistics tools, and physics libraries. Maple Software enables users to solve equations symbolically, visualize mathematical functions, and generate code. Maple Software runs on portable workspaces and dedicated scientific stations. Maple Software is trusted by mathematicians, scientists, engineers, calculus instructors, and symbolic math researchers worldwide.
